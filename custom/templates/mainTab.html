About

Welcome to the GWH(ArcGIS Online/Portal) Administrative Tools application. This application allows administrators to perform key administrative tasks on their ArcGIS Online Org.

This application is in Beta. While all tasks have been tested, no warranty of perfect performance should be inferred.

For more information on this application contact Phil Holleran at pholleran@esri.com. For information regarding use with GWH contact Ian Wilson at iwilson@esri.com.
